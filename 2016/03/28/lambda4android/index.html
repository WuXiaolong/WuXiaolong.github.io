lambda语法简介

基本格式

intput种类

  • 无输入 void
    () -> body

使用 lambda

  • 一个参数输入
    x -> body

使用 lambda

  • 多个参数输入
    (x, y) -> x + y;

使用 lambda

  • 不省略型別
    (int x, int y) -> x + y;

body 种类

  • 什麼都不做
    () -> {}
  • 单行不需要有返回值，单行可省略{}
    (x, y) -> x + y;
  • 单行需要有返回值
    (x, y) -> x + y//注意没有分号结尾

使用 lambda

  • 多行不需要有返回值
    (x, y) ->{
    x x;
    y     y;
    }
  • 多行需要有返回值
    (x, y) ->{
    x x;
    y     y;
    return x + y;
    }
